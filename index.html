مرحباً بك في عالم GitHub!

خطوات إنشاء مستودع جديد:

  1. إنشاء حساب على GitHub إذا لم يكن لديك واحد
  2. النقر على زر "+" في الزاوية اليمنى العليا
  3. اختيار "New repository"
  4. تسمية المستودع وكتابة وصف له (اختياري)
  5. اختيار "Public" أو "Private"
  6. النقر على "Create repository"
ملاحظة: بعد إنشاء المستودع، يمكنك رفع الملفات باستخدام الواجهة الرسومية أو عن طريق Git عبر سطر الأوامر.

أوامر Git الأساسية:

git init
git add .
git commit -m "رسالة التوضيح"
git branch -M main
git remote add origin [رابط المستودع]
git push -u origin main
